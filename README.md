## Install

1. Open VS Code
2. Press F1
3. Type "install"
4. Select "Extensions: Install Extension".
5. Select Thai Lorem Ipsum from the list

## Usage

In your editor , type `thlo` to generate 1 paragraph of lorem ipsum or use `thlo` with number (example : `thlo20`) to generate 20 words .
